Price Formulas

Buy Price
base + distance − city_mod − lang_mod − rel_mod − rep_discount + event_delta
Positive modifier → cheaper to buy. Negative → more expensive.
Sell Price
base + distance + city_mod + lang_mod + rel_mod + event_delta
Positive modifier → higher sell price. Negative → lower.

Each modifier = floor(base × |pct|), applied with sign per direction. Modifiers sum additively before rounding.

Distance Bonus

bonus = base × ((1 + hopPct) ^ hops − 1)
hops = min hops from any producer city. Ecbatana capped at max 2 hops. Coriander, Sesame, Saffron sold in all cities (Spice Merchant) but still accrue distance costs.
